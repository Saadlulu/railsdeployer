railsdeployer
=============

This simple bash script will create a rails ready environment from any ubunutu
server.

Installs:

Essentials Libraries {
	build-essential
	libssl-dev
	libyaml-dev
	libreadline-dev
	openssl
	curl 
	git-core
	zlib1g-dev 
	bison 
	libxml2-dev 
	libxslt1-dev 
	libcurl4-openssl-dev 
	nodejs
	libsqlite3-dev
	gcc
	make
	libmysqlclient-dev
}

Servers {
mysql-server 
mysql-client
libmysqlclient-dev
sqlite3
nginx-extras 
passenger
}

and ofcourse ruby and ruby on rails :)
